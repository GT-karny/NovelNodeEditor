# NovelNodeEditor

NovelNodeEditorは、React Flowを利用してシーンノードを視覚的に編集できるシンプルなエディタです。ノードの追加・削除、ドラッグによる移動、接続操作、およびローカルストレージへの保存・読み込みに対応しています。

## 開発環境のセットアップ

```bash
npm install
```

## 開発サーバーの起動

```bash
npm run dev
```

ブラウザで表示されるURL（通常は http://localhost:5173 ）へアクセスしてください。

## 主な機能

- ノードの追加・削除とドラッグ移動
- 接続ハンドルを使ったエッジの作成と削除（右クリックメニュー）
- フローデータのローカルストレージへの保存 / 読み込み
- ダークテーマのTailwind CSSベーススタイル

## ビルド

```bash
npm run build
```

生成物は`dist/`ディレクトリに出力されます。
